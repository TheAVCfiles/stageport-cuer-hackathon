# IP Boundary

This repository is a public hackathon evaluation surface for the StagePort x CueR use-case instance.

## Public Evaluation Surface

The public repository includes only materials needed for judges and technical reviewers to understand, run, and evaluate the scoped CueR event-floor agent instance:

- Synthetic event/floor data
- Reference agent service
- Cloud Run deployment path
- MCP adapter boundary
- A2A-ready agent card
- Receipt and keychain schemas
- Submission and testing documentation

## Private StagePort Core

The following remain private and are not licensed, disclosed, or transferred through this repository:

- StagePort platform architecture
- StudioOS methodology
- Production room infrastructure
- Commercial scoring and governance logic
- Client or event-specific implementations
- Real attendee, venue, or customer data
- Production signing keys, secrets, credentials, and deployment tokens
- Internal roadmap, prompt systems, strategy, and non-public product language

## Evaluation Use Only

This kit demonstrates the hackathon instance: a live-event introduction and access agent that uses synthetic data, human approval, and proof receipts. It does not grant permission to reproduce the StagePort product or represent derivative work as StagePort, CueR, StudioOS, or Global AVC Systems Inc.
