# Security

Do not commit secrets, API keys, service account JSON, Replit secrets, Google Cloud credentials, private signing keys, production database URLs, or real event/attendee data.

This repository uses synthetic fixtures for public evaluation.

## Public Repository Rule

Anything in this repository should be treated as public, inspectable, forkable, and cloneable. StagePort production infrastructure, private methodology, commercial governance logic, and internal implementation details must remain outside this repository.

## Demo Access

The live demo may be password protected. The demo password belongs only in private hackathon testing instructions, not in this public repository.
