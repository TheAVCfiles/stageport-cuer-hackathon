# Open Source Scope

This repository open-sources only the StagePort x CueR hackathon evaluation instance.

## Included

- CueR reference agent service
- Synthetic event-floor fixtures
- Cloud Run deployment path
- MCP adapter boundary
- A2A-ready agent card
- Receipt and keychain schemas
- Judge testing instructions
- Requirements matrix
- Architecture documentation

## Excluded

- Full StagePort platform
- StudioOS methodology
- Production source code
- Private rooms or client implementations
- Commercial scoring, governance, routing, or safety logic
- Real event, venue, attendee, or client data
- Signing keys, credentials, secrets, or deployment tokens
- Internal product roadmap
- Trademark or brand rights beyond nominative reference

## Intent

The purpose of this repository is to allow judges and technical reviewers to evaluate the StagePort x CueR hackathon submission without disclosing the underlying StagePort product infrastructure or proprietary methodology.

The public code demonstrates the use-case instance. It does not license the broader StagePort system.
